ECG Signal Classifier

The ECG Classifier is an interface designed for classifying ECG signals. It was built using a model trained on six different types of arrhythmias: Bundle Branch Block, Cardiomyopathy, Dysrhythmia, Healthy Controls, Myocardial Infarction, and a category for signals that don't belong to any of the aforementioned types.
